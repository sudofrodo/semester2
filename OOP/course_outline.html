Course Code: CSC241
Course Title: Object Oriented Programming
Credit Hours: 4(3,1)
Corequisite(s):
Prerequisite(s): CSC103

Course Learning Outcomes / Course Objectives :
CLO-1 Demonstrate fundamental principles and concepts of object-oriented programming. Understanding CLO-2 Apply the concepts of object-oriented programming principles (abstraction, inheritance, and polymorphism.) along with interfaces and exception handling to solve a real-world problem. Applying CLO-3 Apply event handling model to develop event-driven programs that respond to user events. Applying CLO’s for Lab CLO-4 Implement a small module utilizing Object-Oriented design. Applying CLO-5 Develop a GUI based project for a real-world problem in a team environment. Creating

Syllabus / Theory Course Outline :
Unit # Topic 1. Overview of Object-Oriented Programming: Evolution, Advantages Principles, Classes & its Concepts: Data & Member Functions; Member Access Specifier; Constructors & Destructor; Memory Allocation; Garbage Collection, and Encapsulation. 2. Object Design and Programming: Abstraction; Inheritance; Polymorphism; Method Overriding; Associations; and Delegation 3. Interfaces & Exception Handling: Overview; Interface Declaration; Implementing an Interface, Interface Inheritance; Standard Exception; Handling Options; Exception Class Hierarchy; Checked vs. Unchecked Exceptions; Catching an Exception System Exceptions vs. Application Exceptions. 4. Collections API: Arrays; Collections Framework; Collections Interfaces; Concrete Collections; Iterating through Collections; Input/Output API: Streams & Files; Input & Output Streams; File Streams; Object Streams; Object Serialization; Readers & Writers. 5. Swing Classes; Events & Event Handlers; Canonical Uses: GUIs, Reactive Framework, and Externally & Program-Generated Events.

Practical Requirements :
This lab emphasizes the concepts of object-oriented techniques used in developing computer-based system. The topics include: Problem solving in Object Oriented Paradigm, Defining classes and objects in JAVA, Controlling access to members- Encapsulation, Passing and returning non-primitive values from methods, Composition/Containership (Has-a relationship), Inheritance, Method Overriding and Abstract Class, Polymorphism, Interfaces, Array List and Generics, File Handling, Graphical User Interface – Layout Managers , Graphical User Interface- Event Driven Programming.
